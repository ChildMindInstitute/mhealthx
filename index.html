mhealthx

Welcome to mhealthx¶

The Child Mind Institute is developing mhealthx as an open source feature extraction pipeline for sensor data. Arno Klein originally created mhealthx at Sage Bionetworks to extract features from mobile health research apps such as mPower, the Parkinson disease symptom tracking app built on top of Apple’s ResearchKit. See our software documentation and Github repository.
